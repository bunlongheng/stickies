# Stickies

> Personal sticky notes board — colorful, draggable, real-time synced.

**Live → [stickies-cyan.vercel.app](https://stickies-cyan.vercel.app)**

![Screenshot](screenshot.png)

---

## Features

- **Colorful sticky notes** — each note gets a hue you can customize
- **Drag & drop** — reposition notes freely on the canvas
- **Real-time sync** — notes persist via Supabase
- **Google sign-in** — secure auth, your notes stay private
- **Search** — filter notes instantly by content
- **List / Board view** — toggle between views
- **Mobile responsive**
- **Dark UI**

## Stack

- Next.js 15 · React 19 · TypeScript
- Supabase (database + auth)
- Google OAuth
- Tailwind CSS · Vercel
